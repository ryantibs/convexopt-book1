\chapter{Bregman divergences}

This chapter studies a concept which generalizes Euclidean distance. This
generalization measures the discrepancy between two points using the curvature
of a given convex function.   

\section{Definition and properties}

\index{Legendre function}
Let $\phi : \R^d \to (-\infty,\infty]$ be a function of Legendre type, which we
recall (Chapter \ref{sec:conjugate_smoothness}) means that $\phi$ is essentially
smooth and essentially strictly convex. The \emph{Bregman divergence} $D_\phi$
corresponding to $\phi$ is a map defined for \smash{$x \in \bar\Omega =
  \dom(\phi)$} and $y \in \Omega = \interior(\dom(\phi))$ by     
\index{Bregman divergence}
\begin{equation}
\label{eq:bregman_divergence}
D_\phi(x,y) = \phi(x) - \phi(y) - \nabla\phi(y)^\T (x - y).
\end{equation}
The value $D_\phi(x,y)$ can be interpreted as the gap between the value
$\phi(x)$ and its first-order approximation $\phi(y) + \nabla\phi(y)^\T (x -
y)$ centered at $y$. As $\phi$ is convex, we know that $D_\phi(x,y) \geq
0$. Moreover, as $\phi$ is strictly convex on the line segment joining $x$ and
$y$, we have $D_\phi(x,y) > 0$ for $x \not= y$.  

We refer to $\phi$ as the \emph{potential function} of the Bregman divergence
$D_\phi$. An important special case is the quadratic 
potential \smash{$\phi(x) = \frac{1}{2} \|x\|_2^2$}. In this case, one can check
that \smash{$D_\phi(x,y) = \frac{1}{2} \|x-y\|_2^2$}, half of the squared
Euclidean distance between $x$ and $y$. In general, we note that $D_\phi$ is not
necessarily a proper distance (or monotone transformation of a distance); it
does not even need to be symmetric in its arguments. Still, Bregman divergences
admit a number of useful properties, described next.     

\paragraph{Convex in the first argument.}
\parlab{par:bregman_first}

For any fixed point $y$, the map $x \mapsto D_\phi(x,y)$ is convex. This is
because such a map can be expressed as $x \mapsto \phi(x) + a^\T x + b$, where
$a,b$ depend on $y$. This means that a Bregman ball centered around the second 
argument, of the form   
\[
\{ x : D_\phi(x,y) \leq r \}
\]
for fixed $y,r$, is a convex set. Furthermore, Bregman balls of this type are
also compact if $\phi$ is closed (Exercise \ref{ex:bregman_first}).     

\paragraph{Star-shaped in the second argument.} 
\parlab{par:bregman_second}

For fixed $x$, the map $y \mapsto D_\phi(x,y)$ is not necessarily
convex. However, it satisfies a weaker property: it is \emph{star-shaped}, where 
$x$ serves as its star-center. To be precise, for any $y$, the map $t \mapsto 
D_\phi(x, (1-t) x + ty)$ is nonincreasing on $[0,1]$ (Exercise
\ref{ex:bregman_second}). Consequently, a Bregman ball centered around the first 
argument,          
\[
\{ y : D_\phi(x,y) \leq r \}
\]
for fixed $x,r$, is what is known as a \emph{star-convex} set: for any $y$ in
this set, the line segment joining $x$ and $y$ lies entirely in the set. 
\index{star-convex set}

\paragraph{Conditions for quadratic growth.}

If $\phi$ is Lipschitz smooth with parameter $L>0$, then it is easy to verify
(using Theorem \ref{thm:lipschitz_smoothness} part (iii)) that 
\index{Bregman divergence!quadratic growth}
\[
D_\phi(x,y) \leq \frac{L}{2} \|x-y\|_2^2.
\]
If $\phi$ is strongly convex with parameter $m>0$, then it is similarly easy to
verify (using Theorem \ref{thm:strong_convexity} part (iii)) that 
\[
D_\phi(x,y) \geq \frac{m}{2} \|x-y\|_2^2.
\]
Therefore, for a Lipschitz smooth and strongly convex potential $\phi$, the
Bregman divergence $D_\phi(x,y)$ grows quadratically in $x-y$ (in the sense that
it is upper and lower bounded by constant multiples of the squared Euclidean
distance between $x$ and $y$).

\paragraph{Three-point identity.}
\parlab{par:bregman_three_point}

Euclidean distance satisfies the following \emph{three-point identity}: 
\begin{equation}
\label{eq:three_point}
\frac{1}{2} \|x-y\|_2^2 = \frac{1}{2} \|x-z\|_2^2 + \frac{1}{2} \|z-y\|_2^2 + 
(z-y)^\T (x-z), 
\end{equation}
for any points $x,y,z \in \R^d$, which can be checked by direct
calculation. This is sometimes called the \emph{law of cosines}. To interpret 
this, note that the third term on the right-hand side is $(z-y)^\T (x-z) =
-\|x-z\|_2 \|y-z\|_2 \cos(\theta)$, where $\theta$ is the angle in between $x-z$  
and $y-z$; thus the above identity relates one side length of a triangle with
vertices $x,y,z$ to the other two side lengths and the cosine of the angle in
between them. (This generalizes the Pythagorean theorem for right triangles,
which corresponds to $\cos(\theta) = 0$.) Bregman divergences admit an
analogous three-point identity:     
\index{Bregman divergence!three-point identity}
\begin{equation}
\label{eq:bregman_three_point}
D_\phi(x,y) = D_\phi(x,z) + D_\phi(z,y) + (\nabla\phi(z) - \nabla\phi(y))^\T
(x-z),  
\end{equation}
for any \smash{$x \in \bar\Omega$} and $y,z \in \Omega$, which can again be
checked by direct calculation. Now the inner product on the right-hand side can
be interpreted as measuring the cosine of the angle in between $x-z$ and
$\nabla\phi(z) - \nabla\phi(y)$, and \eqref{eq:bregman_three_point} can be
understood as a kind of generalized cosine law.  

\medskip

\begin{Example}
A key example of a Bregman divergence is the Kullback-Leibler divergence. This
corresponds to the negative entropy potential function   
\begin{equation}
\label{eq:negative_entropy}
\phi(x) = \sum_{i=1}^d x_i \log(x_i),
\end{equation}
This is Legendre with $\dom(\phi) = \R_+^d$, the nonnegative orthant (we
interpret $0 \log(0) = 0$, by continuous extension). In this case, one can check 
that    
\index{Kullback-Leibler divergence}  
\begin{equation}
\label{eq:kullback_leibler}
D_\phi(x,y) = \sum_{i=1}^d x_i \log(x_i / y_i) + \sum_{i=1}^d (y_i - x_i).
\end{equation}
If $x,y$ lie on the probability simplex, then the above reduces to
\smash{$\sum_{i=1}^d x_i \log(x_i / y_i)$}, called the \emph{Kullback-Leibler 
  divergence} between $x,y$. Observe that \eqref{eq:kullback_leibler} is a
notable case of a Bregman divergence (like squared Euclidean distance) which is
jointly convex in $x,y$ (Exercise \ref{ex:loss_functions} part d), recalling
that in general we only know it is convex in $x$ for fixed $y$. 
\end{Example}

\section{Bregman projection}
\label{sec:bregman_projection}

A Bregman divergence can be used to define an associated projection operator, as 
follows. Let $\phi$ be a closed Legendre potential, let $\Omega =
\interior(\dom(\phi))$, and let $C$ be a closed convex set with $C \cap \Omega
\not= \emptyset$. The \emph{Bregman projection} of a point $y \in \Omega$ onto
$C$ is then defined by         
\index{Bregman projection}
\begin{equation}
\label{eq:bregman_projection}
P^\phi_C(y) = \argmin_{x \in C} \, D_\phi(x,y). 
\end{equation}
Under the stated conditions, this is well-defined, meaning that the minimizer in 
\eqref{eq:bregman_projection} exists and is unique, and furthermore, it holds
that \smash{$P^\phi_C(y) \in \Omega$} (Exercise
\ref{ex:bregman_projection}). The last fact implies that iterated projection is
well-defined. It follows that Bregman projection is idempotent:
\smash{$P^\phi_C(P^\phi_C(y)) = P^\phi_C(y)$}.       

In a variety of ways, Bregman projections generalize Euclidean projections, in
that they exhibit analogous properties. Noting that $\nabla_x D_\phi(x,y) =
\nabla\phi(x) - \nabla\phi(y)$, the first-order optimality condition
\eqref{eq:first_order_optimality} for the problem in
\eqref{eq:bregman_projection} says that, for \smash{$z = P^\phi_C(y)$},   
\index{Bregman projection!variational inequality}
\begin{equation}
\label{eq:bregman_variational_inequality}
(\nabla\phi(z) - \nabla\phi(y))^\T (x - z) \geq 0, \quad \text{for any $x \in
  C$}, 
\end{equation}
which generalizes the variational inequality \eqref{eq:variational_inequality}
for Euclidean projection. The \emph{firmly nonexpansive} property
\eqref{eq:projection_firm_nonexpansiveness1} 
of Euclidean projection has a Bregman generalization as well: denoting 
\smash{$z_x = P^\phi_C(x)$} and \smash{$z_y = P^\phi_C(y)$}, we have 
\index{Bregman projection!firm nonexpansiveness}
\begin{equation}
\label{eq:bregman_firm_nonexpansiveness}
(\nabla\phi(x) - \nabla\phi(y))^\T (z_x - z_y) \geq (\nabla\phi(z_x) -
\nabla\phi(z_y))^\T (z_x - z_y) \geq 0, \quad \text{for all $x,y \in \Omega$},
\end{equation}
which follows from the variational inequality
\eqref{eq:bregman_variational_inequality} (Exercise
\ref{ex:bregman_firm_nonexpansiveness}). Further, a generalization of the
Pythagorean theorem holds for Bregman projections (Exercise
\ref{ex:bregman_pythagorean_theorem}), which can be used to establish
a different type of nonexpansiveness (called quasi-nonexpansiveness). 

From another angle, to understand the differences between Bregman and Euclidean 
projections, we can build intuition as follows. We begin by interpreting the
projection \smash{$z = P^\phi_C(y)$} of $y$ onto $C$ as the point at which the
lowest level contour of the map $x \mapsto D_\phi(x,y)$ intersects $C$. Now
suppose we trace a path which cuts through the contours back to $y$, in the  
steepest way possible: by following the normal direction to
the contours as we decrease the contour level down to zero. Such a normal
direction at any point $x$ is defined by   
\[
-\nabla_x D_\phi(x,y) = \nabla\phi(y) - \nabla\phi(x).
\]
In the case of Euclidean projection, \smash{$\phi(x) = \frac{1}{2} \|x\|_2^2$},
hence this normal direction evolves as a straight line $y-x$ as a function of
$x$. However, for a general potential $\phi$, this direction evolves as a curved 
line $\nabla\phi(y) - \nabla\phi(x)$ in $x$. In other words, while this
constructed path from $y$ to $z$ is a straight line for Euclidean projection, it
is curved for Bregman projection in general. This reflects the fact that Bregman  
divergence $D_\phi(x,y)$ measures the discrepancy between $x,y$ in a geometry 
informed by the curvature of $\phi$. Figure \ref{fig:bregman_projection} gives
an illustration. Exercise \ref{ex:bregman_gradient_flow} formalizes related
ideas.

\begin{Example}
\label{xa:simplex_kl_projection}
We return to the negative entropy potential \eqref{eq:negative_entropy} of the 
last example, and let $C = \{x : 1^\T x = 1, \, x \geq 0\}$, the probability 
simplex. In this case, the Bregman projection of $y$ onto $C$ is (Exercise   
\ref{ex:kullback_leibler_projection}):  
\index{Kullback-Leibler divergence}  
\index{probability simplex!projection}
\begin{equation}
\label{eq:simplex_kl_projection}
P^\phi_C(y) = \frac{y}{\|y\|_1}.
\end{equation}
Note that this is considerably simpler than Euclidean projection onto the
simplex (studied in Exercise \ref{ex:simplex_projection}). Note further that 
we might interpret this result as saying that Kullback-Leibler divergence 
\eqref{eq:kullback_leibler} is to the $\ell_1$ ball (simplex) as Euclidean
distance is to the $\ell_2$ ball: projection in each case reduces to
normalization.
\end{Example}

\section{Conjugacy relations}

Recall that a closed convex function $\phi$ is of Legendre type if and only if
its convex conjugate $\phi^*$ is (Theorem \ref{thm:conjugate_legendre}).
This means that to each Bregman divergence $D_\phi$ (with $\phi$ closed), we can
associate a conjugate divergence \smash{$D_{\phi^*}$}, namely     
\index{Bregman divergence!conjugate divergence}  
\begin{equation}
\label{eq:conjugate_bregman}
D_{\phi^*}(u,v) = \phi^*(u) - \phi^*(v) - \nabla\phi^*(v)^\T (u - v).
\end{equation}
This is defined for $u,v \in \interior(\dom(\phi^*)) =
\interior(\ran(\nabla\phi))$. Now suppose we set
\[
v = \nabla\phi(x) \quad \text{and} \quad u = \nabla\phi(y).
\]
We call $v,u$ the image of $x,y$ in mirror space, respectively, since
$\nabla\phi$ is often called the \emph{mirror map} in the context of Bregman 
divergences. Using $(\nabla\phi)^{-1} = \nabla\phi^*$ (Theorem
\ref{thm:conjugate_legendre}), we also have
\[
x = \nabla\phi^*(v) \quad \text{and} \quad y = \nabla\phi^*(u). 
\]
Since equality holds in Fenchel's inequality for the pairs $x,v$ and $y,u$
(Theorem \ref{thm:conjugate_subgradients}), we have   
\begin{align*}
\phi(x) + \phi^*(v) &= v^\T x, \\
\phi(y) + \phi^*(u) &= u^\T y.
\end{align*}
Combining the relations in the above displays, observe that
\begin{align*}
\phi(x) - \phi(y) - \nabla\phi(y)^\T (x - y) 
&= \phi(x) - u^\T x - \phi(y) + u^\T y \\
&= (v - u)^\T x - \phi^*(v) + \phi^*(u) \\ 
&= \phi^*(u) - \phi^*(v) - \nabla\phi^*(v)^\T (u-v). 
\end{align*}
Through these elementary calculations we have established the following result,
which we codify into a theorem.     

\begin{Theorem}
\label{thm:conjugate_bregman}
For closed convex $\phi,\phi^*$ of Legendre type, and any points $x,y \in 
\interior(\dom(\phi))$, 
\[
D_\phi(x,y) = D_{\phi^*}(\nabla\phi(y), \nabla\phi(x)). 
\]
\end{Theorem}

To summarize, the Bregman divergence between $x,y$ can be formulated in terms of
the conjugate divergence once we map $x,y$ to mirror space (note carefully the
reversal in the order of arguments in $D_\phi$ versus \smash{$D_{\phi^*}$}). An
implication of this result is described next. We will see conjugacy connections 
leveraged again in the coming sections on exponential families and proper
losses.    

\paragraph{Mirror-convex in the second argument.}

We revisit the structure of the map $y \mapsto D_\phi(x,y)$ for fixed 
$x$, through the lens of Theorem \ref{thm:conjugate_bregman}. Assuming $\phi$
is closed and $x \in \interior(\dom(\phi))$, note that we can express this map as  
\[
y \mapsto D_{\phi^*}(\nabla\phi(y), \nabla\phi(x)).
\]
This is \emph{mirror-convex}: the composition of a convex function (as 
$D_{\phi^*}$ is convex in its first argument, by Property
\parref{par:bregman_first}) with the mirror map $\nabla\phi$. 
% Note: mirror map is continuous by Theorem 25.5/26.5 of Rockafellar (1970). 
Similarly, a Bregman divergence ball centered around the second argument can be
expressed as 
\[
\{ y : D_\phi(x,y) \leq r \} = \nabla\phi^*\Big( \big\{ u : D_{\phi^*}(u,
\nabla(\phi(x)) \leq r \big\} \Big).
\]
For fixed $x,r$, the right-hand side is the image of a convex set under the
conjugate mirror map $\nabla\phi^*$. We therefore see that the conjugate
formulation of Bregman divergences provides further insights into their
structure and geometry, beyond the coarser (star-shaped and star-convex)
conclusions in Property \parref{par:bregman_second}.       

\section{Exponential families*}

Exponential families are intimately connected to Bregman divergences, and they
provide us with a helpful language to interpret conjugacy relations from the
last section. Recall (Chapter \ref{sec:maximum_likelihood}) that a regular
exponential family is of the form           
\index{exponential family}
\[
p_\eta(z) = \exp\big(T(z)^\T \eta - \psi(\eta)\big) h(z),
\]
where $\eta \in \R^k$ is called the natural parameter, and $\psi$ is the
log-partition function with $\dom(\psi)$ open. In any such exponential 
family, we know that $\psi$ is convex and differentiable (Exercise
\ref{ex:exponential_family_convex}). In this section, we will further assume
$\psi$ is of Legendre type. This is true for exponential families satisfying a
minimal representation condition (Exercise
\ref{ex:exponential_family_legendre}).       

\subsection{Mirror map as mean map}

In an exponential family, the mirror map in the log-partition function
$\nabla\psi$ has an intuitive interpretation: it is the \emph{mean map},
as it maps the natural parameter $\eta$ to the corresponding mean of the
statistic $T(Z)$ under the distribution $p_\eta$. Precisely,  
\[
\nabla\psi(\eta) = \E_\eta[T(Z)],
\]
where the expectation on the right-hand side is with respect to $Z \sim
p_\eta$. This identity is verified by the calculations in parts c and d of
Exercise \ref{ex:exponential_family_convex}. Denoting $\mu_\eta =
\nabla\psi(\eta)$, below are some common examples (each with identity sufficient  
statistic $T(z) = z$):    
\begin{alignat*}{3}
&\text{Gaussian:} \quad &&\psi(\eta) = \tfrac{1}{2} \eta^2, \quad &&\mu_\eta =
\eta; \\
&\text{Bernoulli:} \quad &&\psi(\eta) = \log(1+e^\eta), \quad &&\mu_\eta =
\tfrac{e^\eta}{1+e^\eta}; \\
&\text{Poisson:} \quad &&\psi(\eta) = e^\eta, \quad &&\mu_\eta = e^\eta. 
\end{alignat*}
Just as $\nabla\psi$ can be seen as the mean map, its inverse $\nabla\psi^{-1} =
\nabla\psi^*$ is the natural parameter map: it takes each mean $\mu$ to the
corresponding natural parameter $\eta_\mu$. For the same three families:      
\begin{alignat*}{3}
&\text{Gaussian:} \quad &&\psi^*(\mu) = \tfrac{1}{2} \mu^2, \quad &&\eta_\mu =
\mu; \\
&\text{Bernoulli:} \quad &&\psi^*(\mu) = \mu\log\mu + (1-\mu) \log(1-\mu), \quad
&&\eta_\mu = \log\hspace{-2pt} \big( \tfrac{\mu}{1-\mu} \big); \\
&\text{Poisson:} \quad &&\psi^*(\eta) = \mu\log\mu - \mu, \quad &&\eta_\mu =
\log(\mu). 
\end{alignat*}

\subsection{Maximum likelihood as Bregman projection}
\label{sec:exponential_family_bregman}

Now let $y_i$, $i=1,\dots,n$ be independent samples, where each $y_i$ is drawn
from a common exponential family distribution (common $T,\psi,h$), but has a
separate natural parameter value $\eta_i \in \R^k$. Let $\eta =
(\eta_1,\dots,\eta_n) \in N$ be a restriction of the parameter space, where $N
\in \R^{nk}$ is a given convex set. For example, in a generalized linear model
(GLM), we would take       
\index{generalized linear model}
\[
N = \Big\{ (\eta_1,\dots,\eta_n) : \eta_i = x_i^\T \beta, \, i=1,\dots,n \;\,
\text{for some $\beta \in \R^d$} \Big\}, 
\]
where $x_i \in \R^d$, $i=1,\dots,n$ are given feature vectors. In what follows,
we will additionally assume $T(y_i) \in \interior(\dom(\psi^*)) =
\interior(\ran(\nabla\psi))$, $i=1,\dots,n$; note that this is a harmless
assumption for some families (Gaussian) but not others (Bernoulli). We make 
this assumption for simplicity of exposition, and the general case
(allowing $T(y_i) \in \boundary(\dom(\psi^*))$) is treated in Exercise 
\ref{ex:exponential_family_boundary}.   

With this setup in place, define $\lambda_i = \nabla\psi^*(T(y_i))$, and compute 
\begin{align*}
D_\psi(\eta_i, \lambda_i) &= \psi(\eta_i) - \psi(\lambda_i) -
  \nabla\psi(\lambda_i)^\T (\eta_i - \lambda_i) \\  
&= \psi(\eta_i) - T(y_i)^\T \eta_i + c_i,
\end{align*}
where $c_i$ denotes a constant not depending on $\eta_i$. This is, up to
constants, the negative log likelihood $-\ell(\eta_i; y_i) =
-\log(p_{\eta_i}(y_i))$. Summing over $i=1,\dots,n$, we obtain    
\[
\sum_{i=1}^n D_\psi(\eta_i, \lambda_i) = -\sum_{i=1}^n \ell(\eta_i; y_i) + c, 
\]
for a constant $c$. In other words, maximum likelihood in this problem is
equivalent to minimizing a sum of Bregman divergences between each natural
parameter $\eta_i$ and $\lambda_i = \nabla\psi^*(T(y_i))$, the result of mapping
of the sufficient statistic $T(y_i)$ to the natural parameter space. 

This can be expressed even more succinctly: defining an aggregate potential
\smash{$\Psi(\eta) = \sum_{i=1}^n \psi(\eta_i)$}, maximum likelihood over $\eta
\in N$ is equivalent to    
\index{exponential family!maximum likelihood estimation}
\begin{equation}
\label{eq:exponential_family_bregman}
\minimize_{\eta \in N} \quad D_{\Psi}\big( \eta, \nabla\Psi^*(T(y)) \big),
\end{equation}
Maximum likelihood in an exponential family can thus be understood as
follows: we map the vector of sufficient statistics $\nabla\Psi^*(T(y)) =
(\nabla\psi^*(T(y_1)), \dots, \nabla\psi^*(T(y_n)))$ to the natural parameter 
space, and then we perform a Bregman projection onto the feasible set $N$.    

\subsection{Maximum likelihood as conjugate Bregman minimization} 

The conjugate formulation of Bregman divergence developed previously provides a
nice way to reinterpret problem \eqref{eq:exponential_family_bregman}. Let
$\mu_\eta = \nabla\Psi(\eta) = (\nabla\psi(\eta_1), \dots, \nabla\psi(\eta_n))$
be the mean vector associated with the natural parameter vector $\eta$. Then, by
Theorem \ref{thm:conjugate_bregman}, maximum likelihood
\eqref{eq:exponential_family_bregman} is equivalent to
\begin{equation}
\label{eq:exponential_family_conjugate}
\minimize_{\eta \in N} \quad D_{\Psi^*}( T(y), \mu_\eta).
\end{equation}
Once again this is highly intuitive: we are minimizing the discrepancy between 
the sufficient statistic vector $T(y)$ and the mean vector $\mu_\eta$, over
feasible parameters $\eta \in N$.  

The conjugate representation of the negative log-likelihood in exponential
families has important statistical implications for studying what is called     
optimism, explored in Exercise \ref{ex:exponential_family_optimism}. 

\section{Proper loss functions*}

The study of what are known as proper losses in probabilistic
forecasting is also deeply connected to Bregman divergences. In this section, we
denote by $y \in \{1,\dots,k\}$ a categorical response variable of interest,
which is unobserved and serves as our target for prediction. We denote by $p$
a probabilistic prediction of $y$, where $p \in \Delta^k = \{ p : 1^\T p = 1, \,
p \geq 0\}$, the simplex in $\R^k$. To be clear, we view each coordinate $p_i$
as a prediction of the probability that $y = i$.         

A loss function $\ell$ is this setting is a map which takes the pair $(p,y)$ to
a real value $\ell(p,y)$, where lower values indicate higher accuracy of the 
probabilistic prediction. Common examples include: 
\begin{alignat*}{3}
&\text{log loss:} \quad &&\ell(p,y) = -\log(p_y); \\
&\text{quadratic loss:} \quad &&\ell(p,y) = -p_y + \frac{1}{2} \|p\|_2^2; \\
&\text{spherical loss:} \quad &&\ell(p,y) = -\frac{p_y}{\|p\|_2}.
\end{alignat*}
A common feature of these example losses is that they are \emph{proper}:
this means that for $y \sim q$ (that is, $\P(y = i) = q_i$, $i=1,\dots,k$), 
the choice $p=q$ minimizes the expected score $\E_q[\ell(p,y)]$ over the
simplex $p \in \Delta^k$. It will be convenient to overload notation and define
$\ell(p,q) = \E_q[\ell(p,y)]$, with the expectation understood as being with
respect to $y \sim q$. Then a proper loss is defined by:
\index{proper loss}
\begin{equation}
\label{eq:proper_loss}
\ell(p,q) \geq \ell(q,q), \quad \text{for all $p,q \in \Delta^k$}. 
\end{equation}
A \emph{strictly proper} loss is one for which the inequality in
\eqref{eq:proper_loss} is strict for $p \not= q$. 

The probabilistic forecasting literature often motivates proper loss functions 
as incentivizing the forecaster to be ``honest''. This can be understood through
subjective probability: suppose $q$ denotes the forecaster's subjective belief
about the response $y$, and $p$ denotes the forecaster's reported belief. Then a
rational forecaster, who seeks to minimize a strictly proper loss $\ell(p,q)$,
always sets $p = q$.     

\subsection{Bregman representation}

Each loss function $\ell$ in the current setting gives rise to what we call a
divergence function, defined as
\[
d(p,q) = \ell(p,q) - \ell(q,q). 
\]
Clearly, a proper loss \eqref{eq:proper_loss} admits a nonnegative divergence:
$d(p,q) \geq 0$, for $p,q \in \Delta^k$.  

A fundamental result is that divergences of proper losses $\ell$ correspond to
Bregman divergences in convex potentials $\phi$. 

\begin{Theorem}
\label{thm:proper_loss_bregman}
Let $\ell : \Delta^k \times \Delta^k \to \R$. Then $\ell$ is proper
\eqref{eq:proper_loss} if and only if its divergence $d$ satisfies for all $p,q
\in \Delta^k$, 
\index{proper loss!Bregman representation}
\begin{equation}
\label{eq:proper_loss_bregman1}
d(p,q) = \phi(q) - \phi(p) - s_p^\T (q-p), \quad \text{with $s_p \in \partial
  \phi(p)$},  
\end{equation}
for a convex function $\phi$ that is subdifferentiable on
$\Delta^k$. Furthermore, a loss is strictly proper if and only if the above
holds for a strictly convex function $\phi$. Finally, when $\phi$ is
differentiable, note that \eqref{eq:proper_loss_bregman1} can be written 
succinctly as 
\begin{equation}
\label{eq:proper_loss_bregman2}
d(p,q) = D_\phi(q,p),
\end{equation}
where $D_\phi$ denotes the Bregman divergence with potential $\phi$, as in 
\eqref{eq:bregman_divergence}. 
\end{Theorem}

The convex potential $\phi$ in Theorem \ref{thm:proper_loss_bregman} that
fulfills the representation \eqref{eq:proper_loss_bregman1} is not mysterious:
it has the explicit form $\phi(p) = -\ell(p,p)$ (unique up to an affine term in
$p$). The proof of this theorem uses elementary arguments in convex analysis and
is outlined in Exercise \ref{ex:proper_loss_bregman}. 

Returning to the previous examples, we can identify the potential functions in
each case:
\begin{alignat*}{3}
&\text{log loss:} \quad &&\phi(p) = \sum_{i=1}^k p_i \log(p_i); \\
&\text{quadratic loss:} \quad &&\phi(p) = \frac{1}{2} \|p\|_2^2; \\
&\text{spherical loss:} \quad &&\phi(p) = \|p\|_2.
\end{alignat*}
These are verified in Exercise \ref{ex:proper_loss_examples}. (Note that,
technically, Theorem \ref{thm:proper_loss_bregman} does not apply to log loss
because this loss takes infinite values on the relative boundary of the 
simplex. An extension of this theorem to such unbounded losses is given in
Exercise \ref{ex:proper_loss_unbounded}.)

\item \label{ex:bregman_gradient_flow}
  This exercise studies the gradient-following path discussed in Chapter
  \ref{sec:bregman_projection} and illustrated in Figure
  \ref{fig:bregman_projection}. Formally, this path is defined the differential 
  equation   
  \[
  x'(t) = - \nabla_x D_\phi(x(t),y), \quad \text{for $t > 0$}, 
  \]
  with the initial condition $x(0) = z$. This is an instance of \emph{gradient 
    flow}. Here $x : \R_+ \to \R^d$ is a curve through which we think of as a
  function of time $t$, and $x'(t)$ denotes its derivative at $t$. Equivalently,
  the gradient flow path of interest is   
  \begin{equation}
  \label{eq:bregman_gradient_flow}
  x'(t) = \nabla\phi(y) - \nabla\phi(x(t)), \quad \text{for $t > 0$}.
  \end{equation}
  In the following parts, we will study this path in a special case in which it
  admits an analytic solution, and compare to the behavior of the solution in a
  related differential equation.  
  
\begin{enumerate}[label=\alph*.]
\item Set $d=2$, $\phi(x) = e^{x_1} + e^{x_2}$, and $y=0$. The gradient flow
  path \eqref{eq:bregman_gradient_flow} becomes 
  \[
  x'_i(t) = 1 - e^{x_i}, \quad \text{for $i=1,2$}.
  \]
  Prove that the solution, subject to the initial condition $x(0) = z$, is 
  \[
  x_i(t) = -\log(1 - a_i e^{-t}), \quad \text{for $i=1,2$}.
  \]
  for constants $a_i = 1-e^{-z_i}$, $i=1,2$. 

\item Eliminate time to derive the following equation for the second coordinate
  of the solution as a function of the first: 
  \[
  x_2 = -\log(\alpha e^{-x_1} + (1-\alpha)),
  \]
  where $\alpha = a_2/a_1$. Verify that this is a curved path whenever $z_1
  \not= z_2$.  

\item Instead of gradient flow, consider the path defined by (assuming twice
  differentiable $\phi$): 
  \[
  x'(t) = - (\nabla^2 \phi(x(t))^{-1} \nabla_x D_\phi(x(t),y), \quad \text{for
    $t > 0$}.
  \]
  This is an instance of what is called \emph{Riemannian gradient flow}. At each
  time $t$, the local geometry is warped by the Hessian of the potential
  function at $x(t)$. Equivalently, 
  \begin{equation} 
  \label{eq:bregman_riemannian_flow}
  x'(t) = (\nabla^2 \phi(x(t))^{-1} \big( \nabla\phi(y) - \nabla\phi(x(t))
  \big), \quad \text{for $t > 0$}.
  \end{equation}
  Prove that in the setting considered in parts a and b, the solution, subject
  to the same initial condition $x(0) = z$, is   
  \[
  x_i(t) = \log(1 + b_i e^{-t}), \quad \text{for $i=1,2$},
  \]
  for constants $b_i = e^{z_i}-1$, $i=1,2$. Eliminate time to derive the
  following expression for the solution:
  \[
  x_2 = \log(\beta e^{x_1} + (1-\beta)),
  \]
  where $\beta = b_2/b_1$. Verify again that this is a curved path whenever $z_1
  \not= z_2$.   

\item The gradient flow \eqref{eq:bregman_gradient_flow} and Riemannian
  gradient flow \eqref{eq:bregman_riemannian_flow} paths do not seem too
  different at this point: they each produce curved paths in the original (primal) 
  space. A key difference is revealed once we map to mirror (dual) space. Let 
  $u(t) = \nabla\phi(x(t))$, that is, $u_i(t) = e^{x_i(t)}$ for $i=1,2$. Prove
  that the gradient flow solution from part b has the mirror space
  representation   
  \[
  \frac{1}{u_2} = \frac{\alpha}{u_1} + 1-\alpha,
  \]
  which is still a curved path (for $\alpha \not= 1$), whereas the Riemannian
  gradient flow solution from part c transforms to
  \[
  u_2 = \beta u_1 + 1-\beta.
  \]
  which is a straight line.

\item Finally, show that this is a general property: Riemannian gradient flow
  is a straight line in mirror space (``dually flat''). Prove using the chain
  rule that \eqref{eq:bregman_riemannian_flow} is the same as
  \[
  u'(t) = \nabla\phi(y) - u(t), \quad \text{for $t > 0$}.
  \]
  whose solution, subject to the same initial condition $u(0) = \nabla\phi(z)$,
  is     
  \[
  u(t) = \nabla\phi(y) + e^{-t} (\nabla\phi(z) - \nabla\phi(y)). 
  \]
  Since all coordinates travel at the same speed, this is a perfectly straight
  line in mirror space connecting $\nabla\phi(z)$ to $\nabla\phi(y)$.
\end{enumerate}

\index{exponential family!optimism}
\item \label{ex:exponential_family_optimism}
  Consider samples $y_i$, $i=1,\dots,n$ as in Chapter
  \ref{sec:exponential_family_bregman}, and for each $i=1,\dots,n$, let $y^*_i$
  be an independent copy of $y_i$. We will view $y_i$, $i=1,\dots,n$ as the
  training set, which we use to form estimates \smash{$\hat\eta_i$},
  $i=1,\dots,n$ of the natural parameters, and we will view $y_i^*$,
  $i=1,\dots,n$ as the test set, which we use to evaluate the estimates.   
  Accordingly, we define the \emph{optimism} of the natural parameter estimates
  by  
  \[
  \mathrm{Opt} = \sum_{i=1}^n \E\big[ \ell(\hat\eta_i; y_i) -
  \ell(\hat\eta_i; y_i^*) \big],
  \]
  where $\ell(\cdot; y_i)$ denotes the log-likelihood function associated
  with sample $y_i$, and similarly for $\ell(\cdot; y_i^*)$. This is typically
  positive: since the natural parameter estimates are fit on the training set 
  (usually by maximum likelihood or a regularized variant), we would expect the 
  in-sample log-likelihood value \smash{$\ell(\hat\eta_i; y_i)$} to be biased 
  upward relative to out-of-sample value \smash{$\ell(\hat\eta_i; y_i^*)$}. 
  
  In what follows, we assume that each \smash{$\hat\eta_i \in \dom(\psi) =
    \interior(\dom(\psi))$} (since $\dom(\psi)$ is open, by assumption), and
  that each $T(y_i), T(y_i^*) \in \dom(\psi^*)$. 

\begin{enumerate}[label=\alph*.]
\item Recalling $\ell(\cdot; y_i)$ denotes the log-likelihood function
  associated with $y_i$, prove that 
  \[
  \E\big[ \ell(\hat\eta_i; y_i) - \ell(\hat\eta_i; y_i^*) \big] = 
  \E\Big[ D_{\psi^*}(T(y_i^*); \hat\mu_i) - D_{\psi^*}(T(y_i); \hat\mu_i)
  \Big], 
  \]
  where \smash{$\hat\mu_i = \nabla\psi(\hat\eta_i)$}. Hint: follow the
  calculation in Exercise \ref{ex:exponential_family_boundary} part a. 

\item Prove by direct calculation:
  \[ 
  \E\Big[ D_{\psi^*}(T(y_i^*); \hat\mu_i) - D_{\psi^*}(T(y_i); \hat\mu_i)
  \Big] = \Cov(T(y_i), \hat\eta_i),
  \]
  and therefore it holds that
  \[
  \mathrm{Opt} = \sum_{i=1}^n \Cov(T(y_i), \hat\eta_i),
  \]
  which is called \emph{Efron's covariance formula} for optimism, due to 
  \cite{efron1986biased}.  

  \smallskip
  Note: this is an important result because the right-hand side above is
  estimable (without needing access to test samples) using resampling methods
  such as the bootstrap.  
\end{enumerate}

\index{Kullback-Leibler divergence}
\item The Kullback-Leibler divergence \eqref{eq:kullback_leibler} can be
  generally defined between any two probability measures $P,Q$ for which $P$ is 
  absolutely continuous with respect to $Q$. This is:
  \[
  \KL(P,Q) = \int \log\bigg( \frac{\d{P}}{\d{Q}} \bigg) \, \d{P}.
  \]
  Consider a generic maximum likelihood problem,
  \[
  \minimize_{\theta \in \Theta} \quad -\sum_{i=1}^n \ell(\theta; Z_i),
  \]
  where $\ell(\theta; z) = \log(p_\theta(z))$ is a family of densities,
  parametrized by $\theta \in \Theta$. Prove that this can be represented as a 
  \emph{right} Kullback-Leibler projection (as opposed to what may be called a 
  \emph{left} Bregman projection, as in \eqref{eq:bregman_projection}), of the 
  form: 
  \[
  \minimize_{P^n_\theta \in \cP^n_\Theta} \quad \KL(P^n, P^n_\theta),
  \]
  where \smash{$P^n = \frac{1}{n} \sum_{i=1}^n \delta_{Z_i}$} is the empirical
  distribution of the data, and
  \[
  \cP^n_\Theta = \bigg\{ P^n_\theta = \frac{\sum_{i=1}^n p_\theta(Z_i)
    \delta_{Z_i}}{\sum_{i=1}^n p_\theta(Z_i)} : \theta \in \Theta \bigg\}
  \]
  is the discretization of the given family of densities to the data sites.

\item \label{ex:proper_loss_bregman}
  We will prove Theorem \ref{thm:proper_loss_bregman}. 

\begin{enumerate}[label=\alph*.]
\item Let $\phi$ be convex, subdifferentiable on $\Delta^k$, and define $d$
  according to \eqref{eq:proper_loss_bregman1}. Prove that $d(p,q) \geq 0$ for
  all $p,q \in \Delta^k$ and $d(q,q) = 0$, hence $\ell$ is proper. 

\item Assume further that $\phi$ is strictly convex. Extend the arguments in the 
  last part to show that $\ell$ is strictly proper. Hint: recall Exercise
  \ref{ex:strict_convexity_subgradient}. 

\item Let $\ell$ be a proper loss. Define $\phi(q) = -\ell(q,q)$. Show that
  $\phi$ is convex, subdifferentiable on $\Delta^k$, and that
  \eqref{eq:proper_loss_bregman1} holds. Hint: write 
  \[
  \phi(q) = -\inf_{p \in \Delta^k} \, \ell(p,q) = \sup_{p \in \Delta^k} \,
  -\ell(p,q).  
  \]
  Note that \smash{$\ell(p,q) = \sum_{i=1}^k \ell(p, i) q_i$} is linear in
  $q$, and so we can use the partial supremum rule, Property
  \parref{par:function_supremum}, to check convexity. In order to check

  then use Property \parref{par:subgradient_supremum} to argue that 
  \[
  s_q = -(\ell(q,1),\dots,\ell(q,k))
  \]
  is a valid subgradient of $\phi$ at each $q \in \Delta^k$.

\end{enumerate}

  \item there is a convex and subdifferentiable function $\phi$ on

\item In this exercise, we will develop a generalization of Theorem
  \ref{thm:lipschitz_smoothness} using Bregman divergences. As before, let
  $\phi$ be Legendre wth $\Omega = \interior(\dom(\phi))$. Let $f : \R^d \to 
  (-\infty, \infty]$ be differentiable with $\dom(f) \supseteq \Omega$, and let 
  $L>0$. Consider the following statements:        
\begin{enumerate}[label=(\roman*)]
\item $\|\nabla f(x) - \nabla f(y)\|_2 \leq L \frac{D_\phi(x,y) + D_\phi(y,x)}
  {\|x-y\|_2}$, for all $x,y \in \Omega$, $x \not= y$;
\item the function $L\phi-f$ is convex on $\Omega$;
\item $f(y) \leq f(x) + \nabla f(x)^\T (y-x) + L D_\phi(y,x)$, for all $x,y \in
  \Omega$; 
\item $(\nabla f(x) - \nabla f(y))^\T (x - y) \leq L (D_\phi(x,y) +
  D_\phi(y,x))$, for all $x,y \in \Omega$. 
\end{enumerate}
Consider the following additional statement (for twice differentiable $f,\phi$):  
\begin{enumerate}
\item[(v)] $\nabla^2 f(x) \preceq L \nabla^2 \phi(x)$, for all $x \in \Omega$. 
\end{enumerate} 
Now we prove various connections between the above statements (mirroring
Exercise \ref{ex:lipschitz_smoothness}).
